input 태그

사용자로부터 입력을 받을 수 있는 입력 필드(input filed)를 정의할 때 사용.

type 속성 값을 달리하여 여러가지 모양을 나타낼 수 있음. MDN 링크

type="text"

type="password"

type="file"

type="button"

type="submit"

type="checkbox"

type="radio"

type="search"

※ input type의 종류는 굉장히 많지만 쓰는 속성은 몇 개 안됨.

label 태그

input의 라벨 역할.

문법 : <label for="input-id">라벨이름</label>

라벨의 for 속성과 input의 id를 연결해 사용.

textarea 태그

여러 줄의 텍스트를 입력할 수 있는 영역.

input:text의 확장판이라고 생각하면 됨.

우하단을 드래그해 입력 공간을 넓힐 수 있는 특징이 있음.

select 태그

드롭다운 리스트를 구성하는 태그.

input:text의 확장판이라고 생각하면 됨.

우하단을 드래그해 입력 공간을 넓힐 수 있는 특징이 있음.

<select name="" id="">

 <option value="">text</option>

</select>

button 태그

클릭 가능한 버튼을 나타내는 태그.

type="submit"

ㄴ 버튼이 서버로 양식을 제출함. 기본값

type="reset"

ㄴ <input type="reset">와 같이 모든 입력을 초기화 함.

type="button"

ㄴ 기본 행동이 없으며 눌렀을 때 아무런 반응을 하지 않음.

ㄴ 스크립트에서 기능을 넣어줄 수 있으며, 우리가 버튼을 대부분 이 용도로 사용함.
