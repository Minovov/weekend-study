img 태그

가장 기본적은 이미지를 만드는 태그

이미지 태그는 인라인 요소입니다.

문법 : <img href="" alt="">

src : 이미지 파일 주소

alt : 이미지 로딩 실패 시(엑박) 보여줄 대체 텍스트

title : 로딩된 이미지에 hover 시 보여줄 텍스트

width

height

picture 태그

HTML5에 새롭게 등장한 다중(반응형) 이미지 태그.

이 태그는 여러 개의 소스 태그(선택)와 이미지 태그(필수)로 구성됩니다.

브라우저에서 각 조건을 체크한 후 부합하는 첫번째 조건의 이미지를 표시합니다.

모든 조건에 부합하지 않는다면 마지막의 이미지 태그를 사용합니다.

픽처 및 소스 태그는 인라인 요소입니다.

문법

<picture>
  <source media="" srcset="">
  <source media="" srcset="">
  <img src="" alt="">
</picture>

<picture> : 이미지를 감싸는 태그. 내부에 소스와 이미지 태그 포함.

<source> : 반응형으로 적용할 이미지. 내부에 조건과 이미지를 표시.

media : 조건. 주로 화면 크기를 설정함

srcset : 이미지의 주소. img 태그의 src 속성과 같음.
