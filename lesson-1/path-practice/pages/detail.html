※ 상대 경로로 작업합니다.
경로 설명

경로 표현: '/', '.', '..' 으로 표현.

  • / : 최상위 폴더 (root)
  • ./ : 현재 폴더
  • ../ : 상위 폴더
  • /폴더명 : 하위 폴더

예시

  • ./script : 현재 html에 있는 script 폴더
  • script : 현재 html에 있는 script 폴더
  • ./script/good.js : script 폴더 안에 있는 good.js 파일
  • script/good.js : script 폴더 안에 있는 good.js 파일
